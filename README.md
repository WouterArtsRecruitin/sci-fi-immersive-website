# 🌌 Sci-Fi Immersive Website

Een cutting-edge WebGL-powered ervaring met floating holographic interfaces,
cinematic particle systems, en experimentele UI interacties.

## 🛠️ Tech Stack

- **React 18** + **TypeScript** - UI framework
- **Three.js** + **React Three Fiber** - 3D rendering
- **Framer Motion** - UI animaties
- **Tailwind CSS** - Styling
- **Zustand** - State management
- **Vite** - Build tool

## 🚀 Quick Start

```bash
# Installeer dependencies
npm install

# Start dev server
npm run dev

# Build voor productie
npm run build
```

## 📁 Project Structuur

```
src/
├── components/
│   ├── Scene/          # Three.js 3D components
│   │   ├── ProceduralEnvironment.tsx
│   │   ├── ParticleSystem.tsx
│   │   ├── CameraController.tsx
│   │   └── GridBackground.tsx
│   └── UI/             # React UI components
│       ├── GlassmorphismCard.tsx
│       ├── NeonText.tsx
│       ├── FloatingButton.tsx
│       └── LoadingScreen.tsx
├── hooks/              # Custom React hooks
├── store/              # Zustand state management
├── utils/              # Utility functions
├── shaders/            # GLSL shaders
└── App.tsx             # Root component
```
